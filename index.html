字符串值无法被修改，字符串对象可以引用别的字符串

字符串变量中保存的是这个值所在的地址（相当于指针）

字符串中的 . 为 “\.” 如果用split(“.”)进行分割字符串时 要加上双反斜杠

声明一个字符串变量

String str = "abcd";
String str = new String("abcd");

输入一个字符串的方法是

impor java.util.Scanner;

Scanner s = new Scanner(System.in);
String str = s.nextLine();

常用的一些使用字符串的方法：

查找字串的位置的方法：

s.indexOf(int c); //字符
s.indexOf(String str);
s.lastIndexOf(String str); //最后子串出现的位置

获取指定位置的字符：

s.charAt(int index);

获取子字符串：

s.substring(int beginIndex);
s.substring(int beginIndex,int endIndex);

字符串判断：

判断字符串的开始和结尾

s.startsWith(String prefix);
s.endsWith(String suffix);

判断字符串是否相等

//可以用 == 进行判断
s.equals(String str);
s.equalsIgnoreCase(String str); //忽略大小写
s.compareTo(String str); //比较字符，字符不同，返回字符的差，如果前几个字符相同 ("si" "sik") 返回长度差

判断是否包含某个子字符串

s.contains(String str);

判断字符串是否为空

s.isEmpty();

空的两种状态

String s = "";   //空字符串
String s = null; //空对象

字母大小写转换

s.toLowerCase();
s.toUpperCase();

字符串分割

s.split(String str);
s.split(String str,int limit);

去除空格(前面和后面的空格，不包括中间的空格)

s.trim();

字符串替换

s.replace(char oldChar,char newChar);
s.replace(String oldStr,String newStr);

Java的产生随机数的方法

import java.util.Random

Random r = new Random;
int ran = r.nextInt(10); //生成一个0-9的数

下午的时候接着昨天的看，感觉 Scheme 这个语言很有魅力啊

这一节练习中要求自己写个温度转换的程序，先去 Google 温度转换公式，再把公式转换成 scheme 语句，2分钟写出来了

(define (Fahrenheit->Celsius f)
    (/ (- f 32) 1.8))

测试数据和网上的在线测试一样的，证明程序是成功了，接着它让我们使用教学包中的函数再次进行测试

(convert-gui Fahrenheit->Celsius)
(convert-repl Fahrenheit->Celsius)
(convert-file "in.dat" Fahrenheit->Celsius "out.dat")

第一个函数功能是Gui界面，创建一个带有按钮和滚动条的窗口；

第二个函数功能是仿真一个交互式窗口，类似命令行，在里面输入数据；

第三个函数功能是把 in.dat 这个文件里面的数值进行转换，并且把文件写到 out.dat 文件中，事先要创建 in.dat 这个文件，并写入数值，空格分隔。

这三个数据没有空格分开，是我人工分的，挖个坑，以后埋。

虽然说用了教学包，但是一行代码把一个功能写出来，更会激起新手极大的兴趣啊。

定义程序convert3，输入三个数，代表个位、十位、百位，程序输出相应的数值

(define (convert3 x y z)
  (+ x (* 10 y) (* 100 z)))

(convert3 1 2 3)     输出结果为321

习题2.3.1 计算毛收入计算所得税编写程序 tax，编写 netpay，计算税后所得，假定每小时工资12美元。

(define (tax wege)
  (* (/ 15.0 100) wege))

(define (wege h)
  (* 12 h))

(define (netpay h)
  (- (wege h) (tax (wege h))))

总结：有没有发现很奇怪的地方，wege 是一个类似函数的东西，却可以作为参数传进去，这让我想起了 python，python 的所有东西都是对象，可以把函数作为参数使用（应该是这样，不对请指正）。

编写 Scheme 程序要遵循一些规则，都是以 “（ ” 开始，接着是操作，然后是复合表达式，最后以 “ ）” 结束。
在笔记（一）里面我猜测 area-of-disk 为函数名，这一节就说了确实是函数名。

Scheme 规定操作先于操作数出现。

设计一个程序至少需要4个步骤：

给每一个程序一个有意义的名字，并且说明输入数据和所产生的数据的类型，称为程序的合约。

;; area-of-ring : number number -> number

注释：;; 表示该行是一个注释。

冒号的左边是程序的名字，右边是输入和输出的类型，输入和输出之间用箭头隔开。

有了合约，就可以加入函数头部，函数头部给每个输入一个不同的名字，这些名字是变量，同时也是程序的参数。

;; area-of-ring : number number -> number
(define (area-of-ring outer inner) ...)

基于合约和参数，简要阐明程序的用途。

;; area-of-ring : number number -> number
;; 计算一个半径为outer，洞半径为inner的圆环面积
(define (area-of-ring outer inner) ...)

整个程序的设计

 ;; 合约: area-of-ring : number number -> number
;; 用途: 计算一个半径为outer，洞半径为inner的圆环面积
;; 例子: (area-of-ring 5 3) 的计算结果为50.24
;; 定义: [函数头部的精化]
(define (area-of-ring outer inner) ...)
  (- (area-of-ring outer)
       (area-of-ring inner)))

;; 测试:
(area-of-ring 5 3)
;; 预期值
50.24

第三章 通过一个小例子对两种程序开发风格进行比较。

一个影院可以定制票价，当票价5美元时，有120人观看，当票价调低了0.1美元后，增加了15人，放映一场电影支付180给供片商，每个人有4美分的开销，想知道利润与票价之间的确切关系，定下一个利润最高的票价。

首先需要分析一下关系:

1.利润是收入和成本之差；

2.收入是票价乘以人数；

3.成本是180美元加4美分乘以人数

4.人数和票价的关系

;; profit : number -> number
;; 对于给定ticket-price，利润=收入-成本
(define (profit ticket-price) ...)

;; revenue : number -> number
;; 对于给定ricket-price，计算收入        
(define (revenue ricket-price) ...)

;; cost : number -> number
;; 对于给定ticket-price，计算支出        
(define (cost ticket-price) ...)

;; attendees : number -> number
;; 对于给定ticket-price，计算观众数        
(define (attendees ticket-price) ...)

接下来完善函数

(define (profit ticket-price)
  (- (revenue ricket-price)                   (cost ticket-price)))

(define (revenue ricket-price)
  (* (attendees ticket-price)                   ticket-price))

(define (cost ticket-price)
  (+ 180 (* 0.4 (attendees ticket-price))))

(define (attendees ticket-price)
  (+ 120 (* (/ 15 .10) (- 5.00 ticket-price))))

总结：对每一种关系使用一个辅助函数，会使结构更加清晰。

linux command

发表于 2019-01-28

man 获取Linux帮助手册

man date 获取date命令的介绍和详细的用法

–help 输出最简单有用的介绍

ls 列出目前工作目录所含子文件及子目录

ls -a 显示所有文件及目录

ls -l 除文件名称外，将文件形态、权限、拥有者、文件大小等咨询详细列出

ls -r 将文件以相反次序显示

ls -t 将文件依建立时间先后次序列出

ls -A 不列出”.”(当前目录）及”..”(父目录

ls -F 在列出的文件名称后加一符号，如可执行档加”*”,目录加”/“

ls -R 目录下有文件，则文件皆依序列出

cd 切换当前工作目录

“~”表示home目录的意思

“.” 表示当前所在目录

“..” 表示当前目录位置的上一层目录

pwd 用于显示所在工作目录的绝对路径名称 ==print working directory==

mkdir 新建文件夹

mkdir -p 当父目录存在时忽略，不存在则建立，用此参数可建立多级文件夹

rmdir 删除空文件夹

rmdir -p 删除子目录，如果子目录删除后，父目录为空，父目录也删除

rm 删除文件或目录

rm -i 删除前注意访问

rm -f 直接删除（包含只读文件

rm -r 删除目录及目录下所有文件

rm -rf 强制删除文件(删库跑路)

cp copy 复制文件或目录

cp -a 复制目录保留链接、文件属性，复制目录下的所有内容

cp -d 复制文件时保留链接（类似快捷方式

cp -f 覆盖已存在的目标文件 不给提示

mv 移动文件与目录 或修改文件与目录名称

centos ps -ef 查看进程

ps -ef | grep

ps 命令用于查看当前正在运行的进程。

grep 是搜索

kill -9 [pid] 关闭进程

du 查看当前目录下的所有文件 ls

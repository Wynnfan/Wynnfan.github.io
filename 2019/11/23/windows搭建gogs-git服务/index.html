windows搭建gogs git服务

发表于 2019-11-23

最近公司又来了一个小伙伴，现在需要团队协作，代码管理，问老板要了服务器去搭建自己公司的git仓库，后面老板舍不得花钱买服务器，搬了台台式机主机充当服务器，操作系统是win7，决定选择gogs搭建git服务。

第一步 去git官网下载git

git官网 选择window平台的git安装包

第二步 去gogs官网

gogs官网 下载window版本的(查看电脑操作系统是多少位的)

32位下载386 zip w/mws

64位下载amd64 zip w/mws 解压

第三步 去nssm官网下载nssm

nssm 下载完成以后，解压，进入对应操作系统位数的nssm文件夹，在当前文件夹，按住shift 右键 在此处打开命令行窗口。

输入命令 nssm install gogs

选择path 找到gogs解压目录中的 install-as-service.bat

一般在 ==scripts\windows== 下

选择完成后点击install service

如果后面有操作错误可以使用nssm remove gogs

注明：nssm用来注册windows服务

第四步 启动gogs服务

win10是右键点击此电脑，选择管理，找到左边服务与应用程序中的服务，找到gogs，点击启动

第四步 启动gogs

进入gogs目录，启动命令行输入 gogs web

此时访问localhost:3000 就进入gogs安装界面，数据库可以选择很多种，如果计算机安装了mysql 那么选择mysql就行，如果什么数据库都没有安装 那么可以使用sqlite


可以设置管理员账号 参考上面图片
安装完成以后就可以注册登录使用了。
